เครื่องหมายการจัดส่ง

#สถานการณ์สถานะเงื่อนไขสีความหมาย
1มีเวลาเหลือยังไม่สั่งระยะเวลาสั่งซื้อ > 14 วัน⚪ เทาสบายๆ
2ใกล้ถึงกำหนดสั่งยังไม่สั่งระยะเวลาสั่งซื้อ < 14 วัน🟡 เหลืองเสี่ยง: ถึงเวลาสั่งซื้อ
3เลยกำหนดสั่งแล้วยังไม่สั่งระยะเวลา < 0🔴 แดงล่าช้า: ไม่ได้สั่งตามเวลา
4สั่งแล้ว มีเวลาเหลือกำลังดำเนินการETA < เริ่มงาน, ระยะเวลา > 14 วัน🔵 น้ำเงินตามแผน
5สั่งแล้ว เวลากระชั้นกำลังดำเนินการETA < เริ่มงาน, ระยะเวลา < 14 วัน🟡 เหลืองเสี่ยง: บัฟเฟอร์น้อย
6สั่งแล้ว ไม่ทันกำลังดำเนินการETA > เริ่มงาน🔴 แดงล่าช้า: วัสดุมาช้า
7อยู่ที่ไซต์อยู่ที่ไซต์—🟢 เขียวพร้อม
8ติดตั้งแล้วติดตั้งแล้ว—🟢 เขียวพร้อม
9งานเท่านั้นใดก็ได้ระยะเวลาผลิต = 0 และระยะเวลาจัดส่ง = 0ไม่มีเครื่องหมายสัญญาจ้างงาน ไม่ใช่การจัดส่งวัสดุ

ยืนยันแล้วตามปกติตรวจสอบระยะเวลาผลิตและจัดส่ง

กำหนดสั่งซื้อ = วันเริ่มงาน − ระยะเวลาผลิต − ระยะเวลาจัดส่ง
ETA คำนวณอัตโนมัติจากขั้นตอนปัจจุบันและระยะเวลาที่เหลือ
ถ้าสัญญามีระยะเวลาผลิต = 0 และระยะเวลาจัดส่ง = 0 — เป็นสัญญาจ้างงาน ไม่แสดงเครื่องหมายจัดส่ง แต่ต้นทุนจะแสดงในแผงงาน หากต้องการเตือนเรื่องการจัดซื้อ (เช่น สี) — ตั้งระยะเวลาจัดส่ง > 0
